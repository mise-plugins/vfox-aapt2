# vfox-aapt2

A [vfox](https://github.com/version-fox/vfox) plugin for [aapt2](https://developer.android.com/tools/aapt2) (Android Asset Packaging Tool 2).

## Installation

```bash
vfox add aapt2
```

Or with mise:

```bash
mise use aapt2@latest
```

## Usage

```bash
# List available versions
vfox search aapt2

# Install a specific version
vfox install aapt2@8.7.3

# Use a version
vfox use aapt2@8.7.3
```

## About aapt2

aapt2 (Android Asset Packaging Tool 2) is a build tool that Android Studio and Android Gradle Plugin use to compile and package your app's resources.
